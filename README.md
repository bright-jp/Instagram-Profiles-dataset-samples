この **Instagram Profiles Dataset Sample** には **1,556件のレコード** が含まれており、**Bright Data API** を使用して抽出されました。

## Data Points

| Field Name | Description |
|------------|-------------|
| `account` | Instagramアカウントのユーザー名です。 |
| `fbid` | Instagramアカウントに関連付けられたFacebook IDです。 |
| `id` | Instagramアカウントの一意の識別子です。 |
| `followers` | アカウントのフォロワー数です。 |
| `posts_count` | アカウントが投稿した総投稿数です。 |
| `is_business_account` | アカウントがビジネスアカウントかどうかを示します。 |
| `is_professional_account` | アカウントがプロフェッショナルアカウントかどうかを示します。 |
| `is_verified` | アカウントがInstagramにより認証済みかどうかを示します。 |
| `avg_engagement` | アカウント投稿の平均エンゲージメント率です。 |
| `external_url` | アカウントのbioにリンクされている外部URLです。 |
| `biography` | アカウントプロフィールの自己紹介（bio）セクションです。 |
| `business_category_name` | アカウントがビジネスの場合のビジネスカテゴリ名です。 |
| `category_name` | プロフィールに関連付けられた一般的なカテゴリ名です。 |
| `post_hashtags` | アカウント投稿で使用されているハッシュタグです。 |
| `following` | プロフィールがフォローしているアカウント数です。 |
| `posts` | アカウントが行った投稿の詳細です。 |
| `profile_image_link` | プロフィール画像へのURLリンクです。 |
| `profile_url` | InstagramプロフィールへのURLリンクです。 |
| `profile_name` | Instagramプロフィールの表示名です。 |
| `highlights_count` | プロフィール上のストーリーハイライト数です。 |
| `highlights` | プロフィール上のストーリーハイライトの詳細です。 |
| `full_name` | アカウント所有者のフルネームです。 |
| `is_private` | プロフィールが非公開かどうかを示します。 |
| `bio_hashtags` | プロフィールの自己紹介（bio）に含まれるハッシュタグです。 |
| `url` | プロフィールの自己紹介（bio）セクション内のURLです。 |
| `is_joined_recently` | アカウントが最近作成されたかどうかを示します。 |
| `has_channel` | アカウントにIGTVチャンネルがあるかどうかを示します。 |
| `partner_id` | アカウントのビジネスパートナーに関連付けられたIDです。 |
| `business_address` | アカウントがビジネスの場合の事業所住所です。 |
| `related_accounts` | プロフィールに関連または類似するアカウントです。 |
| `email_address` | アカウントに関連付けられたメールアドレスです。 |
| `external_url_title` | 外部URLのタイトルまたは説明です。 |


## Use Cases

### 1. ソーシャルメディアのインフルエンサー分析
フォロワー数、エンゲージメント率、コンテンツカテゴリを調べることで、潜在的なインフルエンサーを特定して分析します。

### 2. 市場調査とトレンド
プロフィールデータを使用して市場トレンドを追跡し、消費者行動を理解し、製品のポジショニング戦略を改善します。

### 3. 競合ベンチマーキング
Instagram上でのプレゼンス、コンテンツ戦略、オーディエンスのエンゲージメントを分析して、競合をベンチマークします。
